# Diagnóstico
## Campos detectados
- diagnostico
- cie
- presuntivo
- definitivo
## Objetivo

Registrar diagnósticos clínicos.

## Estructura recomendada
diagnosticos: [
  {
    descripcion,
    cie10,
    tipo
  }
]
## Tipos permitidos
- presuntivo
- definitivo
## Restricción

Debe permitir múltiples diagnósticos.

## Preparación futura

Compatibilidad con:

- CIE10 oficial
